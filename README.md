# Warehouse QR Code Management System

## 📌 Description

Warehouse QR Code Management System is a Java-based console application designed to manage inventory efficiently using QR codes. It allows users to add products, generate unique QR identifiers, and retrieve product details using SQLite database integration.

## 🚀 Features

* Add Product with QR Code
* Search Product using QR
* SQLite Database Integration

## 🛠 Technologies Used

* Java
* JDBC
* SQLite

## ▶️ How to Run

1. Import project into Eclipse
2. Add sqlite-jdbc JAR file
3. Run WarehouseApp.java
